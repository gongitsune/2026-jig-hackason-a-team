# 突発単語会話 - タスク一覧

仕様書・実装状況から洗い出したタスク一覧。Googleスプレッドシートにコピペする場合は、下部の「タブ区切り（コピペ用）」セクションを使用してください。

---

## タスクの凡例

### タグ（複数可、カンマ区切り）

| タグ | 説明 |
|------|------|
| バックエンド | API・DB・ビジネスロジック |
| フロントエンド | 画面・UI・JS連携 |
| インフラ | デプロイ・環境構築 |
| 状態遷移・同期 | ルーム状態、全員完了検知、ポーリング、画面遷移 |
| 仕様補足 | 仕様書に明記されていないが必要な項目 |

| 優先度 | 説明 |
|--------|------|
| 高 | ゲーム成立に必須 |
| 中 | 体験向上・重要 |
| 低 | あれば良い |
